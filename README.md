# pyhash
Collection of various hash functions. Python library in progress.

## Hash functions

- [x] BSD 16-bit checksum
- [x] Fletcher checksum
- [x] Adler-32
- [ ] CRC32
- [x] MD2
- [x] MD4
- [x] MD5
- [ ] Jenkins
- [ ] BLAKE
- [ ] BLAKE2
- [ ] RIPEMD series
- [x] SHA-1
- [x] SHA-2
- [ ] SHA-3
- [ ] Whirlpool
